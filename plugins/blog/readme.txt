// Installation

1. Upload the blog folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin
5. Go to your administration panel and setup blog settings
6. In user-group you can setup access for each user-group
7. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the blog plugin
2. Delete the blog folder in the plugins directory over FTP.
3. Done
