// Installation

1. Upload the urlmapping folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin
5. Create old / new urls and in the administration panel - url mapping section
6. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the urlmapping plugin
2. Delete the urlmapping folder in the plugins directory over FTP.
3. Done
