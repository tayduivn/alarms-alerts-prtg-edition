// Installation

1. Upload the gallery folder into the plugins folder
3. Login into your administration panel
4. Go to Plugins/Hooks
5. Install the plugin
6. Set permission to upload folder inside the gallery plugin 0777. (plugins/gallery/upload/)
7. Go to your administration panel and setup gallery settings
8. In user-group you can setup access for each user-group
9. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the gallery plugin
2. Delete the gallery folder in the plugins directory over FTP.
4. Done
