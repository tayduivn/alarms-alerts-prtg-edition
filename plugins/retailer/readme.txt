// Installation

1. Upload the retailer folder into the plugins folder
3. Login into your administration panel
4. Go to Plugins/Hooks
5. Install the plugin
6. Go to your administration panel and setup retailer settings
7. In user-group you can setup access for each user-group
8. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the retailer plugin
2. Delete the retailer folder in the plugins directory over FTP.
4. Done
