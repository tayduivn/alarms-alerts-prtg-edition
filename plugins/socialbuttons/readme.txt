// Installation

1. Upload the social buttons folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin
5. Go to your administration panel and setup buttons in the settings
6. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the social buttons plugin
2. Delete the social buttons folder in the plugins directory over FTP.
3. Done
