// Installation

1. Upload all folders to your server
2. yourtemplate folder is actually your template you use (for example: jakcms)
3. Login into your administration panel
5. Install the Plugin
6. Setup your ticket plugin
7. Don' forget to set the right permissions in the user-groups
8. CHMOD 777 for folder attach in the plugin/ticketing directory
9. Done

// Uninstallation

1. Go to plugins find Ticket Plugin and hit delete
3. Delete all files over FTP
4. Done
