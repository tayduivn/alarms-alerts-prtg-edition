// Installation

1. Upload the belowheader folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin
5. Create content with should be placed below the header and choose pages/news to display
6. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the belowheader plugin
2. Delete the belowheader folder in the plugins directory over FTP.
3. Done
