// Installation

1. Upload the download folder into the plugins folder
3. Login into your administration panel
4. Go to Plugins/Hooks
5. Install the plugin
6. Create a download directory on your server, where you will store your files.
7. Go to your administration panel and setup download settings
8. In user-group you can setup access for each user-group
9. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the download plugin
2. Delete the download folder in the plugins directory over FTP.
4. Done
