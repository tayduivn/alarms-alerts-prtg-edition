// Installation

1. Upload the eCommerce folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin
5. Go to your administration panel and setup your online shop.
6. In user-group you can setup access for each user-group
7. Done

// Edit payment.html template to your needs

// Uninstallation

1. Go to plugins/hooks and uninstall the eCommerce plugin
2. Delete the eCommerce folder in the plugins directory over FTP (optional).
3. Done
