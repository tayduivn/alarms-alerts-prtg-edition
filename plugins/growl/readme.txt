// Installation

1. Upload the growl folder into the plugins folder
2. Login into your administration panel
3. Go to Plugins/Hooks
4. Install the plugin
5. Create a site tour and connect them to your page
6. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the site tour plugin
2. Delete the site tour folder in the plugins directory over FTP.
3. Done
