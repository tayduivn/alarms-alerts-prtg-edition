// Installation

1. Upload the xml_seo folder into the plugins folder
3. Login into your administration panel
4. Go to Plugins/Hooks
5. Install the plugin
6. plugins/xml_seo/files needs chmod 0777
7. Create your xml file in the administration panel and link it to Google/Yahoo/Bing and any others.
8. Done

// Uninstallation

1. Go to plugins/hooks and uninstall the xml_seo plugin
2. Delete the xml_seo folder in the plugins directory over FTP.
4. Done
